# HiCat Movies 🎥

HiCat Movies is a premium, cinematic streaming platform simulator built with HTML, Vanilla CSS, and JavaScript. It leverages the TMDB API to provide a seamless experience for discovering movies and TV shows.

## ✨ Features

- **Premium UI/UX**: A modern, dark-themed interface with smooth transitions and glassmorphism.
- **Dynamic Content**: Explore trending, top-rated, and upcoming movies and TV shows.
- **My Library**: Save your favorite titles to a personal collection (stored locally).
- **⏳ Time Machine**: A nostalgic mode with retro themes for discovering classics.
- **✨ AI Picks**: Personalized recommendations powered by simulated AI logic.
- **Interactive Search**: Real-time search for movies, TV shows, and people.
- **Responsive Design**: Optimized for desktop and mobile viewing.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/puneet821/hicatwatch-platform.git
   ```
2. **Open `index.html`**: Simply open the `index.html` file in your preferred browser.
3. **API Key**: You will need a TMDB API key to fetch live data. Follow the on-screen instructions to set it up.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **API**: [The Movie Database (TMDB)](https://www.themoviedb.org/)
- **Libraries**: PeerJS (for connectivity features)
